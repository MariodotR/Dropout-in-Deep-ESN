# Project Overview

This repository contains Jupyter notebooks focused on deep Echo State Networks (ESNs) with a specific focus on dropout regularization techniques.

## Jupyter Notebooks Descriptions:

1. **Forecasting_Manufacturers_Dropout (2).ipynb**  
   A notebook focused on forecasting manufacturer dropout using deep Echo State Networks with dropout regularization techniques.

2. **Internal.ipynb**  
   A comprehensive notebook containing internal analysis, model development, and experimentation with deep ESN architectures and dropout mechanisms.
